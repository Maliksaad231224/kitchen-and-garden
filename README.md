## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open browser and navigate to
# http://localhost:3000
```

### Build for Production

```bash
npm run build
npm start
```
